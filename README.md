# Enigmatica 2 Expert 1.86版本简体中文汉化补丁

基于发布在[CFPA](https://modpack.cfpa.team/)上由aptx73翻译的1.62版本和hujtt移植至1.81a版本汉化更新而来

同时融入了少量来自1.86+版本的更新内容，例如[New Creative Battery](https://github.com/EnigmaticaModpacks/Enigmatica2Expert/commit/7514d625080030e932175c46f81e8cea4d2597f3)，此次更新修改了玩家可以获得的创造模式能源，修改后的创造模式电容库能提供更高的输入输出流量，优化终局游戏游玩体验

同时建议额外安装[1.91更新](https://github.com/EnigmaticaModpacks/Enigmatica2Expert/blob/master/changelogs/CHANGELOG_MODS_1.91.md)新加入的模组[Finite Gas](https://www.curseforge.com/minecraft/mc-mods/finite-gas)，这个模组修复了高级发电机模组的复制燃气bug
